操作ヘルプ

CubeRackStudio でよく使う操作の一覧です。

  • ジョイントをクリック … 選択中の Beam / Pole をここに追加
  • Alt + クリック … コネクタ（Beam / Pole / Joint）を削除
  • Shift + クリック … そのジョイント位置の脚の ON / OFF
  • マウス中ドラッグ … 回転
  • マウス右ドラッグ … 平行移動（パン）
  • ホイール … ズーム
  • Ctrl + Z … Undo
  • Ctrl + Y … Redo
